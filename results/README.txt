Results Directory
=================

This folder contains all output files generated by the PageRank algorithm.

Files:
-------
- rankings.csv      : Final node rankings with scores and percentages
- convergence.png   : Convergence plot showing rank evolution over iterations
- sensitivity.png   : Sensitivity analysis plot for damping factor variations

Output Format:
--------------
rankings.csv:
  Node,Rank,Score
  0,0.428571,42.86%
  1,0.285714,28.57%
  2,0.285714,28.57%

Usage:
-------
Results are automatically generated when running main.py or analysis.ipynb

Notes:
-------
- PNG images are saved at 300 DPI
- CSV files can be opened with any spreadsheet software
- All outputs are overwritten on each new run
